# 🚀 CPU Scheduling Visualizer

An interactive web-based simulator to visualize CPU Scheduling Algorithms.

## 📌 Features

* FCFS, SJF, Round Robin
* Gantt Chart visualization
* Waiting Time & Turnaround Time
* Context Switching
* Interactive UI

## 🚀 Run Project

Open index.html using Live Server.

## 🧪 Sample Input

P1 → AT=0, BT=20
P2 → AT=1, BT=2
P3 → AT=2, BT=2
P4 → AT=3, BT=2

Quantum = 2

## 🧠 Learning

This project helps understand scheduling algorithms visually.
